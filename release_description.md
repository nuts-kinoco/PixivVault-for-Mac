PixivVault v1.0.0 をリリースしました。

【主な機能と特徴】
- Pixivユーザーの作品（イラスト・マンガ）を一括でローカルに保存できるバックアップツールです。
- 一度ダウンロードした作品はスキップする「差分ダウンロード」に対応。
- 作者がPixiv上で削除・非公開にした作品を検知する機能。
- GUI（Flet）による直感的な操作と進行状況の表示。
- Windows向けに単一の実行ファイル（.exe）としてビルドされています。

【使い方】
1. 添付の `PixivVault.zip` をダウンロードし、解凍して `PixivVault.exe` を取り出します。
2. ブラウザ拡張機能などでエクスポートした `cookies.txt` を用意します。
3. `PixivVault.exe` を起動し、右上の設定（歯車アイコン）から `cookies.txt` を読み込ませます。
4. 保存したい作者のPixivユーザーIDを入力し、「実行」をクリックしてください。

※本ツールは個人の利用範囲内で、サーバーに負荷をかけないようマナーを守ってご利用ください。
