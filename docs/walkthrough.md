# PixivVault Phase 1 (v2.0) ウォークスルー

ご主人様、Phase 1の実装が完了いたしました！
以下の機能がPixivVaultに組み込まれました。

## 📦 追加された機能

### 1. Zip化（アーカイブ）機能の追加
ダウンロード完了時に、作者ごとの画像フォルダを自動的に `.zip` に圧縮する機能を実装しました。
再ダウンロード時や差分更新時でも「全て解凍して再圧縮」するのではなく、**新しい画像だけをZipの末尾に追記（Append）**するスマートな仕様になっています。これにより動作が非常に軽量です。

**UIの変更点:**
- **個別設定**: フォロー一覧の各ユーザーに「📦（アーカイブ）」ボタンを追加しました。青く点灯している作者だけが自動Zip化の対象になります。
- **全体設定**: 設定ダイアログの下部に「Advanced / 高度な設定」セクションを折りたたみで追加しました。ここから「すべての作者に対しダウンロード完了後zipにする」を一括でオンにできます。

### 2. 残り時間（ETA）の推定表示
ダウンロードの進捗状況から処理速度（件/秒）を計算し、「残り約◯◯分◯◯秒」といった目安時間をリアルタイムで表示するようにしました。

**UIの変更点:**
- 個別ダウンロード・一括ダウンロードの両方で、プログレスバーの右側に青文字で残り時間が表示されるようになりました。

## 🛠️ 確認方法（テスト手順）

以下のコマンドでアプリを起動し、動作をご確認ください。
```powershell
python main.py
```

**確認ポイント:**
1. **UIの確認**: フォロー一覧タブを開き、各作者の横に「📦」ボタンがあること、また設定タブの下部に「Advanced」項目が追加されていることを確認してください。
2. **Zip化のテスト**:
   - 任意の作者の「📦」ボタンをオンにするか、Advanced設定で全体Zip化をオンにします。
   - ダウンロードを実行します。
   - ダウンロード完了後、`Images/` フォルダ内に `作者名.zip` が生成され、一時フォルダが削除されていることを確認してください。
3. **残り時間表示のテスト**:
   - ダウンロード中、プログレスバーの横に「残り約◯◯分◯◯秒」または「残り時間計算中...」と表示されることを確認してください。
